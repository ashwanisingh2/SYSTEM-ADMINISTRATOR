UNIT 1: Foundations

CH-01: Computer Hardware & Architecture

#hardware#beginner

Learning Objectives

  • Motherboard components identify karna
  • Storage types compare karna (HDD vs SSD vs NVMe)
  • BIOS/UEFI configure karna
  • Hardware ticket L1 level pe resolve karna

1.1 Motherboard Architecture

TIP

Motherboard ek bada junction board hai jahan computer ka har ek part aakar connect hota hai.

Chipset, slots, aur form factors samajhna ek sysadmin ke liye pehli seedhi hai.

1.2 Storage Deep Dive

Interfaces, speeds, and RAID basics.

INFO

HDD mechanical hoti hai, SSD chip based hoti hai. NVMe PCIe lanes use karti hai.

1.6 Hardware Troubleshooting

DANGER

Common Mistake: PC start nahi ho raha aur display cables motherboard mein lagi hain discrete GPU ki jagah.

Commands Learned

CommandDescription
wmic diskdrive get model,sizeGet disk info (Windows)
chkdsk C: /f /rCheck disk errors
mdsched.exeMemory diagnostic
msinfo32System information

Review Questions

SSD uses flash memory resulting in much higher IOPS and sequential speeds compared to spinning magnetic disks in HDDs.

UEFI supports GUI, secure boot, drives larger than 2TB (GPT), while legacy BIOS uses MBR and is 16-bit text only.

Check debug LEDs, listen to beep codes, reseat RAM, clear CMOS, check PSU connections.

RAID 0 (speed, 0 redundancy). RAID 1 (mirror, 1 disk failure allowed). RAID 5 (striping+parity, 1 disk failure allowed, good balance of capacity and safety).

Windows 10/11 administration tools jo har L1/L2 engineer ko aane chahiye — real tickets ke saath.

UNIT 2: Operating Systems

CH-05: Linux & RHEL Administration

#linux#rhcsa

5.1 Linux Administration Basics

SUCCESS

Linux is the backbone of modern cloud infrastructure.

Commands Learned

# Live logs
journalctl -u service -f

# Open ports
ss -tulnp

# Disk usage
df -h && du -sh /*

# Permissions
chmod 755 file
chown user:group file

# LVM volume
lvcreate -L 10G -n lv vg

Review Questions

755 gives read/write/execute to owner, and read/execute to others. 644 gives read/write to owner, and read-only to others.

Systemd uses parallel startup and units, while SysVinit uses sequential shell scripts.

Use semanage, setsebool, and check /var/log/audit/audit.log or use audit2allow.

CH-06: Windows Server 2022

#windows-server#md-102

6.1 Windows Server Introduction & Roles

Understanding DNS, DHCP, DFS, WSUS, and IIS roles.

WARNING

Always configure static IP before promoting a server to a Domain Controller.

Review Questions

A scope is a single subnet. A superscope groups multiple scopes (subnets) together to provide IPs to a single physical network segment.

Forward resolves Hostname to IP. Reverse resolves IP to Hostname (using PTR records).

To test patches on a small group first before deploying them to the entire production environment.

Minimum 3 drives are required for RAID 5 to store data and distributed parity.
